Capital Punishment.

To string up a man for the very worst crime, Is like smashing a watch for not keeping good time. 397 It were wiser, no doubt To use the bad man in a work of some good And so build up a moral sublime; And, by cleaning it out, To make the bad watch keep the best time we could, This is Truth, though in rhyme.
