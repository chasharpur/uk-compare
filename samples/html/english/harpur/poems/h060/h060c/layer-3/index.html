Capital Punishment.

To string up a man for the very worst crime, Is like smashing a watch for not keeping good time. It were wiser, no doubt To use the bad man in a work of some good, And so build up a moral sublime; While we made the bad watch keep the best time we could, By repairing it well:—this is Truth, though in rhyme.
